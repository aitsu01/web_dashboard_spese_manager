# Web Dashboard Spese Manager

Dashboard web responsive per la gestione delle spese personali, realizzata come progetto finale con **HTML5**, **CSS3**, **Bootstrap 5.3** e **JavaScript**.

## Obiettivo del progetto

L'applicazione permette all'utente di:

- aggiungere una nuova spesa
- visualizzare le spese nella dashboard
- modificare una spesa esistente
- eliminare una spesa
- filtrare le spese per categoria
- cercare una spesa per descrizione
- visualizzare un riepilogo dinamico
- consultare un grafico delle spese per categoria

## Tecnologie utilizzate

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript
- DOM Manipulation
- localStorage
- Chart.js
- Git / GitHub
